# クレカ比較ラボ 運営マニュアル

クレジットカード比較アフィリエイトサイトの自動運営システムです。
このフォルダ1つでサイトの生成・更新・公開がすべて完結します。

---

## 📁 フォルダ構成

```
affiliate-site/
├─ data/
│   └─ cards.json        ← ★ここを編集するだけで全ページが変わる（データの司令塔）
├─ generate.py           ← 全HTMLを自動生成するエンジン
├─ deploy.ps1            ← 再生成＋GitHubへ公開（毎日自動実行される）
├─ setup_github.ps1      ← 初回のGitHub公開セットアップ
├─ style.css             ← デザイン
├─ index.html ほか       ← 自動生成されたページ（直接編集しない）
└─ 運営マニュアル.md      ← このファイル
```

> **重要**: `index.html` や `cards/*.html` などのHTMLは `generate.py` が自動で作ります。
> 直接編集しても次回の自動更新で上書きされます。**編集するのは `data/cards.json` だけ**です。

## 🚀 収益化までの3ステップ

### ステップ1: ASPに登録する（あなた本人が行う作業）

ASP（アフィリエイト・サービス・プロバイダ）に登録し、各カードの広告リンクを取得します。
**本人確認・銀行口座登録が必要なため、この作業はあなた自身が行う必要があります。**

おすすめASP（すべて無料・クレカ案件に強い）:

| ASP | 特徴 | URL |
|-----|------|-----|
| **A8.net** | 国内最大手。審査なしで即登録、クレカ案件多数 | https://www.a8.net/ |
| **もしもアフィリエイト** | 初心者向け、W報酬制度 | https://af.moshimo.com/ |
| **afb（アフィb）** | 金融・クレカ案件に強い | https://www.afi-b.com/ |
| **バリューコマース** | Yahoo!系、大手案件多数 | https://www.valuecommerce.ne.jp/ |

登録手順:
1. 上記サイトでメール登録 → 本人情報・銀行口座を入力
2. 「クレジットカード」で広告（プログラム）を検索
3. 提携申請 → 承認されたら「広告リンク（URL）」をコピー

### ステップ2: 取得したリンクを貼る

`data/cards.json` を開き、各カードの `"affiliate_url": "#"` の `#` の部分を、
ASPで取得したあなた専用のリンクURLに差し替えます。

```json
{
  "id": "rakuten-card",
  "name": "楽天カード",
  ...
  "affiliate_url": "https://px.a8.net/svt/ejp?a8mat=あなた専用のID...",  ← ここを差し替え
  ...
}
```

保存したら、以下を実行して反映:
```powershell
pwsh -File deploy.ps1
```

### ステップ3: サイトを公開する（GitHub Pages・無料）

1. https://github.com/ で無料アカウントを作成
2. 新しいリポジトリを作成（例: `credit-card-lab`）※**Public** にする
3. PowerShellで初回セットアップ（リポジトリURLは作成時に表示されるもの）:
   ```powershell
   pwsh -File setup_github.ps1 -RepoUrl "https://github.com/あなたのユーザー名/credit-card-lab.git"
   ```
4. GitHubのリポジトリ → **Settings → Pages** → Branch を `main / (root)` にして Save
5. 数分後、`https://あなたのユーザー名.github.io/credit-card-lab/` で公開されます🎉

## 🔄 自動更新の仕組み（設定済み）

Windowsタスクスケジューラに **`credit_card_site_daily_update`** というタスクを登録済みです。

- **毎日 朝6:00** に自動で `deploy.ps1` を実行
- サイトを再生成して「最終更新日」を当日に更新（SEOで重要な"鮮度"を保つ）
- GitHubに自動push → 公開サイトに自動反映

> PCが起動していれば自動で動きます。停止したい場合はタスクスケジューラで
> `credit_card_site_daily_update` を無効化してください。

## ✏️ コンテンツの編集方法（すべて data/cards.json で完結）

### カードを追加する
`data/cards.json` の `"cards"` 配列に新しいオブジェクトを追加するだけ。
ランキング・比較表・詳細ページが自動で作られます。

### カードの順位を変える
`"cards"` 配列の並び順 = ランキング順位 です。上に書いたカードほど上位表示されます。
（報酬の高いカード・成約しやすいカードを上位に置くのが収益化のコツ）

### 記事を追加する（SEO対策）
`"articles"` 配列に追加すると、記事ページと記事一覧に自動で反映されます。
検索流入を増やすには「クレジットカード おすすめ」など検索されるキーワードを
タイトルに入れた記事を増やしていくのが効果的です。

### 編集後は必ず実行
```powershell
pwsh -File deploy.ps1
```

## 💡 収益を伸ばすコツ

1. **記事を増やす** — 検索流入がアクセスの9割。月2〜4本ペースで記事を追加
2. **上位カードを最適化** — 報酬が高く・申込されやすいカードをランキング上位に
3. **タイトルにキーワード** — 「年会費無料」「学生」「即日発行」など検索される語を入れる
4. **Googleサーチコンソール登録** — `sitemap.xml` を登録すると検索エンジンに早く認識される
5. **継続が命** — アフィリエイトは3〜6ヶ月で成果が出始める。毎日の自動更新が効いてくる

## ⚖️ 法律・規約の注意

- 各ページに**アフィリエイト広告表示**を記載済み（景品表示法ステマ規制対応）
- カードの還元率・特典は変わるので、`data/cards.json` を定期的に最新情報へ更新
- 誇大表現（「絶対お得」等）は避ける。ASPの規約違反になります
- 申込リンクには `rel="nofollow sponsored"` を自動付与済み（Google推奨）
